# Keyword System

This file defines the keyword groups used to discover jade trend samples on Xiaohongshu.

## How to Use the Keyword System

- Search the groups separately instead of mixing everything into one large query set.
- Keep direct jade queries as the base layer.
- Add modern design and material-combination queries to surface newer product language.
- Add adjacent-category queries to identify style signals that may migrate into jade.

## Search Rules

- Use the exact phrases as seed keywords first.
- Expand with close variants after the first pass.
- For jade price-band research, include colloquial market language as well as numeric price phrases.
- Keep one keyword group in one collection batch when possible.
- Prioritize queries that reveal product images, not just text-heavy notes.

## Filtering Rules

Keep notes that show:

- a clear product form,
- enough visual detail to judge material and finish,
- a meaningful interaction signal,
- relevance to the 3000-8000 RMB price band or adjacent entry points.

Drop notes that are:

- purely inspirational with no product specificity,
- too generic to identify materials or structure,
- duplicative across the same account without adding a new motif.
